## 模块简介

可作为以后所有maven项目的子模块使用，包含一些常用的自定义方法及工具类，减少不同项目中的重复代码。

## 开发原则

> 所提供接口应尽量避免业务逻辑，专注于技术需求

> 原则上不允许在父仓库中对子模块进行修改
